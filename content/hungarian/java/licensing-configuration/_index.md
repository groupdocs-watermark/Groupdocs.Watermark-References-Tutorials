# GroupDocs.Watermark for Java licencelés és konfigurációs útmutatók

A GroupDocs.Watermark for Java egy erőteljes könyvtár, amely lehetővé teszi a fejlesztők számára, hogy programozottan hozzáadjanak, szerkesszenek és kezeljenek vízjeleket különféle dokumentumformátumokon. Több mint 70 fájlformátumot támogat, akár 2 GB méretű dokumentumokat is feldolgoz anélkül, hogy az egész fájlt a memóriába töltené, és 99,9 %-os üzemidő SLA-t biztosít, így megbízható választás vállalati szintű alkalmazásokhoz.

## Hogyan állítsuk be a licencet a GroupDocs.Watermark for Java-ban?

A `License` osztály egy GroupDocs.Watermark licencet képvisel, és a használatakor aktiválja a termék teljes funkcionalitását. Töltsd be a licencfájlt a `new License().setLicense("path/to/license.lic")` paranccsal, vagy adj meg egy `InputStream`‑et a `setLicense(InputStream)` metódusnak. Ez az egyetlen lépés aktiválja a teljes API‑t, eltávolítja a kiértékelési vízjeleket, és feloldja az összes prémium funkciót, például a nagy felbontású renderelést és a kötegelt feldolgozást. A metódus ugyanúgy működik, függetlenül attól, hogy a licenc helyben tárolt, erőforrásokba beágyazott vagy egy távoli szolgáltatásból lekért.
