## Domande frequenti

**D: Come aggiungo una semplice filigrana di testo a un PDF?**  
R: Usa la classe `WatermarkOptions` per definire il testo, il carattere, la dimensione, il colore e la posizione, quindi applicala con il metodo `Watermarker.add`.

**D: Posso aggiungere una filigrana di immagine con trasparenza?**  
R: Sì, l'API ti consente di caricare un'immagine, impostare il livello di opacità e posizionarla ovunque nella pagina.

**D: È possibile bloccare una filigrana in modo che gli utenti non possano eliminarla?**  
R: Assolutamente—abilita la funzione di blocco della filigrana durante la creazione per renderla anti‑manomissione.

**D: GroupDocs.Watermark supporta PDF protetti da password?**  
R: Sì, puoi aprire un PDF protetto fornendo la password in `LoadOptions` prima di applicare le filigrane.

**D: Quali versioni di Java sono compatibili con GroupDocs.Watermark?**  
R: La libreria supporta Java 8 e versioni successive, inclusi Java 11, 17 e le successive versioni LTS.
