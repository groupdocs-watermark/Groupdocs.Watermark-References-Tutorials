# GroupDocs.Watermark per Java – Licenze e Configurazione Tutorial

GroupDocs.Watermark per Java è una libreria potente che consente agli sviluppatori di aggiungere, modificare e gestire filigrane su vari formati di documento in modo programmatico. Supporta oltre 70 formati di file, elabora documenti fino a 2 GB senza caricare l'intero file in memoria e offre un SLA di disponibilità del 99,9 %, rendendola una scelta affidabile per applicazioni di livello enterprise.

## Come impostare la licenza in GroupDocs.Watermark per Java?

La classe `License` rappresenta una licenza GroupDocs.Watermark e attiva la piena funzionalità del prodotto quando applicata. Carica il tuo file di licenza con `new License().setLicense("path/to/license.lic")` o fornisci un `InputStream` a `setLicense(InputStream)`. Questo singolo passaggio attiva l'intera API, rimuove le filigrane di valutazione e sblocca tutte le funzionalità premium come il rendering ad alta risoluzione e l'elaborazione batch. Il metodo funziona allo stesso modo sia che la licenza sia memorizzata localmente, incorporata nelle risorse o recuperata da un servizio remoto.
