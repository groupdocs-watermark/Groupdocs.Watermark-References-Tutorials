# Imposta la licenza GroupDocs Java – Guida alla licenza Watermark

Se stai integrando **GroupDocs.Watermark** in un'applicazione Java, il primo passo è **impostare correttamente la licenza GroupDocs Java**. Senza una licenza valida, l'API funziona in modalità di prova, il che limita le funzionalità e aggiunge filigrane ai documenti elaborati. Questa guida spiega perché la licenza è importante, descrive i vantaggi di una corretta configurazione e ti indirizza ai tutorial dettagliati di cui avrai bisogno per avviarti rapidamente.

## Perché è importante impostare la licenza GroupDocs Java?

Impostare la licenza non solo sblocca l'intero set di funzionalità — come il rendering di filigrane ad alta risoluzione, l'elaborazione batch e il supporto per un'ampia gamma di formati di file — ma garantisce anche la conformità ai termini di licenza di GroupDocs. Una licenza corretta migliora le prestazioni, rimuove le filigrane di prova e ti dà accesso al supporto premium.
