## よくある質問

**Q: PDF にシンプルなテキスト透かしを追加するにはどうすればよいですか？**  
A: `WatermarkOptions` クラスでテキスト、フォント、サイズ、色、位置を定義し、`Watermarker.add` メソッドで適用します。

**Q: 透明度付きの画像透かしを追加できますか？**  
A: はい、API で画像を読み込み、透明度レベルを設定し、ページ上の任意の位置に配置できます。

**Q: ユーザーが削除できないように透かしをロックできますか？**  
A: もちろんです—透かし作成時にロック機能を有効にすれば、改ざん防止透かしになります。

**Q: GroupDocs.Watermark はパスワード保護された PDF をサポートしていますか？**  
A: はい、`LoadOptions` にパスワードを指定すれば、透かし適用前に保護された PDF を開くことができます。

**Q: どの Java バージョンが GroupDocs.Watermark と互換性がありますか？**  
A: ライブラリは Java 8 以降をサポートしており、Java 11、17 などの LTS リリースでも動作します。
