# GroupDocs.Watermark for Java のライセンスと構成チュートリアル

GroupDocs.Watermark for Java は、開発者がさまざまなドキュメント形式に対して透かしをプログラムで追加、編集、管理できる強力なライブラリです。70 以上のファイル形式をサポートし、ファイル全体をメモリに読み込むことなく最大 2 GB のドキュメントを処理し、99.9 % の稼働時間 SLA を提供するため、エンタープライズ向けアプリケーションに信頼できる選択肢となります。

## GroupDocs.Watermark for Java でライセンスを設定する方法

`License` クラスは GroupDocs.Watermark のライセンスを表し、適用すると製品の全機能が有効化されます。`new License().setLicense("path/to/license.lic")` でライセンスファイルをロードするか、`setLicense(InputStream)` に `InputStream` を渡してください。この単一の手順でフル API が有効になり、評価用透かしが削除され、高解像度レンダリングやバッチ処理などのすべてのプレミアム機能が利用可能になります。ライセンスがローカルに保存されている場合、リソースに埋め込まれている場合、またはリモートサービスから取得される場合でも、同じように機能します。
