**Q: ウォーターマークをロックするとドキュメントサイズに影響しますか？**  
A: 影響は最小オーバーヘッドしか追加しません。

**Q: 後でウォーターマークのロックを解除できますか？**  
A: 同じライセンスを使用した API 経由でのみ可能です。エンドユーザーが手動でロックを解除することはできません。
