## クイック概要

- **Primary focus:** Excel ワークブックに効率的に透かしを付ける方法。  
- **Key capabilities:** テキストまたは画像の透かしを追加、シートごとの透かしをカスタマイズ、ヘッダー/フッターの管理、チャート背景の調整。  
- **Why GroupDocs.Watermark?** 流暢な API を提供し、主要なすべての Excel フォーマットをサポートし、既存のワークブック機能を保持しながら安全な透かしを適用します。

## Excel 透かし付けに GroupDocs.Watermark を使用する理由

- **Cross‑platform reliability:** 追加のネイティブ依存関係なしで Windows、Linux、macOS 上で動作します。  
- **Full‑feature support:** `.xls`、`.xlsx`、マクロ有効ファイルを処理し、数式、チャート、データ検証をそのまま保持します。  
- **Security‑first design:** 透かしは文書を破壊せずに削除しにくい形で埋め込まれます。
