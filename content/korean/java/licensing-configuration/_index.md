# GroupDocs.Watermark for Java 라이선스 및 구성 튜토리얼

GroupDocs.Watermark for Java는 개발자가 다양한 문서 형식에 워터마크를 프로그래밍 방식으로 추가, 편집 및 관리할 수 있게 해주는 강력한 라이브러리입니다. 70개 이상의 파일 형식을 지원하고, 전체 파일을 메모리에 로드하지 않고도 최대 2 GB 문서를 처리하며, 99.9 % 가동 시간 SLA를 제공하여 엔터프라이즈 급 애플리케이션에 신뢰할 수 있는 선택입니다.

## GroupDocs.Watermark for Java에서 라이선스 설정 방법

`License` 클래스는 GroupDocs.Watermark 라이선스를 나타내며 적용 시 전체 제품 기능을 활성화합니다. 라이선스 파일은 `new License().setLicense("path/to/license.lic")` 로 로드하거나 `setLicense(InputStream)`에 `InputStream`을 제공하십시오. 이 한 단계로 전체 API가 활성화되고 평가용 워터마크가 제거되며 고해상도 렌더링 및 배치 처리와 같은 모든 프리미엄 기능이 잠금 해제됩니다. 라이선스가 로컬에 저장되든, 리소스에 포함되든, 원격 서비스에서 가져오든 메서드 동작은 동일합니다.
