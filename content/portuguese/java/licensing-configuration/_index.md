# Definir Licença GroupDocs Java – Guia de Licenciamento do Watermark

Se você está integrando **GroupDocs.Watermark** em uma aplicação Java, o primeiro passo é **definir a licença GroupDocs Java** corretamente. Sem uma licença válida, a API funciona em modo de avaliação, o que limita funcionalidades e adiciona marcas d'água aos documentos processados. Este guia explica por que o licenciamento é importante, descreve os benefícios da configuração correta e indica os tutoriais detalhados que você precisará para começar rapidamente.

## Por que definir a licença GroupDocs Java é importante?

Definir a licença não apenas desbloqueia o conjunto completo de recursos — como renderização de marca d'água em alta resolução, processamento em lote e suporte a uma ampla variedade de formatos de arquivo — mas também garante a conformidade com os termos de licenciamento da GroupDocs. O licenciamento adequado melhora o desempenho, remove as marcas d'água de avaliação e oferece acesso ao suporte premium.
