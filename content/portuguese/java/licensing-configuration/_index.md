# GroupDocs.Watermark for Java Licenciamento e Tutoriais de Configuração

GroupDocs.Watermark for Java é uma biblioteca poderosa que permite aos desenvolvedores adicionar, editar e gerenciar marcas d'água em vários formatos de documento programaticamente. Ela suporta mais de 70 formatos de arquivo, processa documentos de até 2 GB sem carregar o arquivo inteiro na memória, e oferece um SLA de disponibilidade de 99,9 %, tornando-a uma escolha confiável para aplicações de nível empresarial.

## Como Definir a Licença no GroupDocs.Watermark for Java?

A classe `License` representa uma licença do GroupDocs.Watermark e ativa a funcionalidade completa do produto quando aplicada. Carregue seu arquivo de licença com `new License().setLicense("path/to/license.lic")` ou forneça um `InputStream` para `setLicense(InputStream)`. Esta única etapa ativa a API completa, remove as marcas d'água de avaliação e desbloqueia todos os recursos premium, como renderização em alta resolução e processamento em lote. O método funciona da mesma forma, independentemente de a licença estar armazenada localmente, incorporada nos recursos ou recuperada de um serviço remoto.
