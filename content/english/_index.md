# Document Watermarking with GroupDocs.Watermark – Complete Guide

GroupDocs.Watermark makes **document watermarking with GroupDocs.Watermark** fast, reliable, and fully customizable for a wide range of file types. Whether you’re building a .NET service that stamps confidential PDFs or a Java web app that brands every PowerPoint slide, this guide shows you exactly how to protect and brand your documents using a single, consistent API.

## Quick Answers
- **What does GroupDocs.Watermark do?** It adds, searches, modifies, and removes text or image watermarks across more than 100 document formats.  
- **Which platforms are supported?** Both .NET (Framework, .NET Core, .NET 5/6) and Java (JDK 8+).  
- **Do I need a license?** A free trial is available; a commercial license is required for production use.  
- **Can I watermark PDFs, Word, and Excel files?** Yes – the API handles PDF, DOCX, XLSX, PPTX, images, and many more.  
- **Is batch processing possible?** Absolutely – you can apply the same watermark to dozens of files in one call.

## What is Document Watermarking with GroupDocs.Watermark?
Document watermarking with GroupDocs.Watermark refers to the process of embedding visible or invisible markers—such as text, logos, QR codes, or signatures—directly into a document’s content stream. These markers help enforce **digital document security**, reinforce brand identity, and deter unauthorized distribution.

## Why Use GroupDocs.Watermark for Document Watermarking?
- **Unified API** – One set of methods works for PDFs, Word documents, Excel spreadsheets, PowerPoint presentations, images, and more.  
- **Rich watermark types** – Text, image, logo, signature, QR code, and even custom shapes.  
- **Precise control** – Position, rotation, opacity, size, and layering are fully configurable.  
- **Tamper‑resistant** – Locked watermarks survive attempts to delete or edit them.  
- **Batch operations** – Process multiple files in a single request, saving time and resources.
