## Häufig gestellte Fragen

**Q: Wie füge ich ein einfaches Textwasserzeichen zu einem PDF hinzu?**  
A: Verwenden Sie die Klasse `WatermarkOptions`, um Text, Schriftart, Größe, Farbe und Position zu definieren, und wenden Sie es dann mit der Methode `Watermarker.add` an.

**Q: Kann ich ein Bildwasserzeichen mit Transparenz hinzufügen?**  
A: Ja, die API ermöglicht das Laden eines Bildes, das Festlegen der Deckkraft und die Positionierung an beliebiger Stelle auf der Seite.

**Q: Ist es möglich, ein Wasserzeichen zu sperren, sodass Benutzer es nicht löschen können?**  
A: Absolut—aktivieren Sie die Wasserzeichen‑Sperrungsfunktion beim Erstellen des Wasserzeichens, um es manipulationsresistent zu machen.

**Q: Unterstützt GroupDocs.Watermark passwortgeschützte PDFs?**  
A: Ja, Sie können ein geschütztes PDF öffnen, indem Sie das Passwort in den `LoadOptions` angeben, bevor Sie Wasserzeichen anwenden.

**Q: Welche Java‑Versionen sind mit GroupDocs.Watermark kompatibel?**  
A: Die Bibliothek unterstützt Java 8 und neuer, einschließlich Java 11, 17 und späteren LTS‑Versionen.
