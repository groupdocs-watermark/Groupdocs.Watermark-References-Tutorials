# GroupDocs.Watermark für Java Lizenzierungs- und Konfigurations-Tutorials

GroupDocs.Watermark für Java ist eine leistungsstarke Bibliothek, die Entwicklern ermöglicht, Wasserzeichen zu verschiedenen Dokumentformaten programmgesteuert hinzuzufügen, zu bearbeiten und zu verwalten. Sie unterstützt über 70 Dateiformate, verarbeitet Dokumente bis zu 2 GB, ohne die gesamte Datei in den Speicher zu laden, und bietet ein SLA mit 99,9 % Verfügbarkeit, was sie zu einer zuverlässigen Wahl für Unternehmensanwendungen macht.

## Wie man die Lizenz in GroupDocs.Watermark für Java festlegt?

Die `License`-Klasse repräsentiert eine GroupDocs.Watermark-Lizenz und aktiviert die vollständige Produktfunktionalität, wenn sie angewendet wird. Laden Sie Ihre Lizenzdatei mit `new License().setLicense("path/to/license.lic")` oder übergeben Sie einen `InputStream` an `setLicense(InputStream)`. Dieser einzelne Schritt aktiviert die gesamte API, entfernt Evaluationswasserzeichen und schaltet alle Premium‑Funktionen frei, wie hochauflösendes Rendering und Batch‑Verarbeitung. Die Methode funktioniert identisch, egal ob die Lizenz lokal gespeichert, in Ressourcen eingebettet oder von einem Remote‑Dienst abgerufen wird.
