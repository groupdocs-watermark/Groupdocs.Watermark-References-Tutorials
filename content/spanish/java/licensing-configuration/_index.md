# Tutoriales de Licenciamiento y Configuración de GroupDocs.Watermark para Java

GroupDocs.Watermark para Java es una biblioteca potente que permite a los desarrolladores agregar, editar y administrar marcas de agua en varios formatos de documento de forma programática. Soporta más de 70 formatos de archivo, procesa documentos de hasta 2 GB sin cargar todo el archivo en memoria, y ofrece un SLA de disponibilidad del 99,9 %, lo que la convierte en una opción confiable para aplicaciones de nivel empresarial.

## Cómo establecer la licencia en GroupDocs.Watermark para Java?

La clase `License` representa una licencia de GroupDocs.Watermark y activa la funcionalidad completa del producto cuando se aplica. Cargue su archivo de licencia con `new License().setLicense("path/to/license.lic")` o proporcione un `InputStream` a `setLicense(InputStream)`. Este único paso activa la API completa, elimina las marcas de agua de evaluación y desbloquea todas las funciones premium, como la renderización de alta resolución y el procesamiento por lotes. El método funciona de la misma manera ya sea que la licencia esté almacenada localmente, incrustada en recursos o recuperada de un servicio remoto.
