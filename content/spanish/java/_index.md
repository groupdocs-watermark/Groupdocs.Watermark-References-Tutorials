## Preguntas frecuentes

**P: ¿Cómo agrego una marca de agua de texto simple a un PDF?**  
R: Usa la clase `WatermarkOptions` para definir el texto, la fuente, el tamaño, el color y la posición, luego aplícala con el método `Watermarker.add`.

**P: ¿Puedo agregar una marca de agua de imagen con transparencia?**  
R: Sí, la API permite cargar una imagen, establecer el nivel de opacidad y posicionarla en cualquier parte de la página.

**P: ¿Es posible bloquear una marca de agua para que los usuarios no puedan eliminarla?**  
R: Absolutamente—activa la función de bloqueo de marcas de agua al crearla para que sea a prueba de manipulaciones.

**P: ¿GroupDocs.Watermark admite PDFs protegidos con contraseña?**  
R: Sí, puedes abrir un PDF protegido proporcionando la contraseña en `LoadOptions` antes de aplicar marcas de agua.

**P: ¿Qué versiones de Java son compatibles con GroupDocs.Watermark?**  
R: La biblioteca es compatible con Java 8 y versiones posteriores, incluyendo Java 11, 17 y futuras versiones LTS.
