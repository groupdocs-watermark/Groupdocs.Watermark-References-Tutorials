## 常見問題

**Q: 如何在 PDF 中加入簡單的文字浮水印？**  
A: 使用 `WatermarkOptions` 類別定義文字、字型、大小、顏色與位置，然後透過 `Watermarker.add` 方法套用。

**Q: 能否加入具透明度的圖片浮水印？**  
A: 可以，API 允許載入圖片、設定不透明度，並將其放置於頁面的任意位置。

**Q: 是否可以鎖定浮水印，使使用者無法刪除？**  
A: 絕對可以——在建立浮水印時啟用鎖定功能，即可讓浮水印具備防篡改特性。

**Q: GroupDocs.Watermark 是否支援受密碼保護的 PDF？**  
A: 支需在 `LoadOptions` 中提供密碼，即可在套用浮水印前開啟受保護的 PDF。

**Q: 哪些 Java 版本與 GroupDocs.Watermark 相容？**  
A: 此函式庫支援 Java 8 及以上版本，包括 Java 11、17 以及後續的 LTS 版本。
