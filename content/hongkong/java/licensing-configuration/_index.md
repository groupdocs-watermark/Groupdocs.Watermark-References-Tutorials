# GroupDocs.Watermark for Java 授權與設定教學

## 如何在 GroupDocs.Watermark for Java 中設定授權？

`License` 類別代表 GroupDocs.Watermark 授權，套用後會啟用完整產品功能。使用 `new License().setLicense("path/to/license.lic")` 載入授權檔，或將 `InputStream` 傳入 `setLicense(InputStream)`。此一步即可啟用完整 API、移除評估水印，並解鎖所有高階功能，如高解析度渲染與批次處理。無論授權檔是本機存放、嵌入資源，或從遠端服務取得，使用方式皆相同。
