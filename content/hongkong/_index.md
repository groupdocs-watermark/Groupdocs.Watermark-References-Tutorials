# 使用 GroupDocs.Watermark 進行文件水印 – 完整指南

GroupDocs.Watermark 讓 **使用 GroupDocs.Watermark 進行文件水印** 變得快速、可靠且可完全自訂，支援各種檔案類型。無論您是建立在 .NET 上的服務以在機密 PDF 上蓋章，或是使用 Java 的 Web 應用程式為每張 PowerPoint 投影片加上品牌標誌，本指南將一步步說明如何使用單一且一致的 API 來保護與品牌化您的文件。

## 快速解答
- **GroupDocs.Watermark 的功能是什麼？** 它可以在超過 100 種文件格式中新增、搜尋、修改與移除文字或圖片水印。  
- **支援哪些平台？** 同時支援 .NET（Framework、.NET Core、.NET 5/6）與 Java（JDK 8+）。  
- **需要授權嗎？** 提供免費試用版；正式上線使用需購買商業授權。  
- **可以為 PDF、Word 與 Excel 檔案加水印嗎？** 可以——API 支援 PDF、DOCX、XLSX、PPTX、圖片等多種格式。  
- **支援批次處理嗎？** 當然可以——您可以在一次呼叫中將相同的水印套用至數十個檔案。

## 什麼是使用 GroupDocs.Watermark 進行文件水印？
使用 GroupDocs.Watermark 進行文件水印是指將可見或不可見的標記（例如文字、標誌、QR 代碼或簽名）直接嵌入文件內容流的過程。這些標記有助於加強 **數位文件安全**、強化品牌形象，並阻止未經授權的散布。

## 為什麼要使用 GroupDocs.Watermark 進行文件水印？
- **統一的 API** – 同一套方法即可處理 PDF、Word 文件、Excel 試算表、PowerPoint 簡報、圖片等多種檔案。  
- **豐富的水印類型** – 文字、圖片、標誌、簽名、QR 代碼，甚至自訂形狀。  
- **精確的控制** – 位置、旋轉、透明度、大小與層次皆可完整設定。  
- **防篡改** – 鎖定的水印能抵抗刪除或編輯的嘗試。  
- **批次操作** – 在單一請求中處理多個檔案，節省時間與資源。

5. **批次處理** – 高效地為多個文件套用水印。  
6. **水印管理** – 搜尋、修改或移除現有的水印。  
7. **跨平台相容性** – .NET 與 Java 平台使用相同的 API。

A: 可以，API 完全相容於 ASP.NET Core、Azure Functions、AWS Lambda 以及 Java Spring Boot 服務。

**Q: 此函式庫支援加密的 PDF 嗎？**
