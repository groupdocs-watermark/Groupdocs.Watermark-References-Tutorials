Dengan mengekstrak ukuran halaman lebih awal, Anda dapat menentukan penempatan watermark yang optimal, menskalakan grafik secara tepat, atau menghasilkan laporan berbasis ukuran. Tutorial berikut memandu Anda melalui setiap langkah—dari ekstraksi metadata dasar hingga penanganan dokumen kompleks dengan banyak bagian.
