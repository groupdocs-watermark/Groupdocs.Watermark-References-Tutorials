# Tutorial Lisensi dan Konfigurasi GroupDocs.Watermark untuk Java

GroupDocs.Watermark for Java adalah pustaka yang kuat yang memungkinkan pengembang untuk menambahkan, mengedit, dan mengelola watermark pada berbagai format dokumen secara programatik. Ini mendukung lebih dari 70 format file, memproses dokumen hingga 2 GB tanpa memuat seluruh file ke memori, dan menyediakan SLA uptime 99,9 %, menjadikannya pilihan yang dapat diandalkan untuk aplikasi tingkat perusahaan.

## Cara Mengatur Lisensi di GroupDocs.Watermark untuk Java?

Kelas `License` mewakili lisensi GroupDocs.Watermark dan mengaktifkan fungsionalitas produk penuh saat diterapkan. Muat file lisensi Anda dengan `new License().setLicense("path/to/license.lic")` atau sediakan `InputStream` ke `setLicense(InputStream)`. Langkah tunggal ini mengaktifkan API penuh, menghapus watermark evaluasi, dan membuka semua fitur premium seperti rendering resolusi tinggi dan pemrosesan batch. Metode ini bekerja sama baiknya apakah lisensi disimpan secara lokal, disematkan dalam sumber daya, atau diambil dari layanan remote.
