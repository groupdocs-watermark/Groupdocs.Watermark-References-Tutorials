## Pertanyaan yang Sering Diajukan

**Q: Bagaimana cara menambahkan watermark teks sederhana ke PDF?**  
A: Gunakan kelas `WatermarkOptions` untuk mendefinisikan teks, font, ukuran, warna, dan posisi, kemudian terapkan dengan metode `Watermarker.add`.

**Q: Bisakah saya menambahkan watermark gambar dengan transparansi?**  
A: Ya, API memungkinkan Anda memuat gambar, mengatur tingkat opasitas, dan menempatkannya di mana saja pada halaman.

**Q: Apakah memungkinkan mengunci watermark sehingga pengguna tidak dapat menghapusnya?**  
A: Tentu—aktifkan fitur penguncian watermark saat membuat watermark untuk membuatnya tahan manipulasi.

**Q: Apakah GroupDocs.Watermark mendukung PDF yang dilindungi kata sandi?**  
A: Ya, Anda dapat membuka PDF yang dilindungi dengan memberikan kata sandi pada `LoadOptions` sebelum menerapkan watermark.

**Q: Versi Java apa yang kompatibel dengan GroupDocs.Watermark?**  
A: Perpustakaan ini mendukung Java 8 dan yang lebih baru, termasuk Java 11, 17, dan rilis LTS selanjutnya.
