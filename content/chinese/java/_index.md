## 常见问题

**问：如何向 PDF 添加简单的文本水印？**  
A: 使用 `WatermarkOptions` 类定义文本、字体、大小、颜色和位置，然后使用 `Watermarker.add` 方法应用。

**问：我可以添加带透明度的图像水印吗？**  
A: 可以，API 允许您加载图像、设置不透明度级别，并将其定位在页面的任意位置。

**问：是否可以锁定水印，使用户无法删除？**  
A: 当然——在创建水印时启用水印锁定功能，使其具备防篡改特性。

**问：GroupDocs.Watermark 是否支持受密码保护的 PDF？**  
A: 是的，您可以在 `LoadOptions` 中提供密码，以打开受保护的 PDF，然后再应用水印。

**问：哪些 Java 版本与 GroupDocs.Watermark 兼容？**  
A: 该库支持 Java 8 及更高版本，包括 Java 11、17 以及后续的 LTS 版本。
