# Οδηγοί Αδειοδότησης και Διαμόρφωσης του GroupDocs.Watermark για Java

Το GroupDocs.Watermark για Java είναι μια ισχυρή βιβλιοθήκη που επιτρέπει στους προγραμματιστές να προσθέτουν, να επεξεργάζονται και να διαχειρίζονται υδατογραφήματα σε διάφορες μορφές εγγράφων προγραμματιστικά. Υποστηρίζει πάνω από 70 μορφές αρχείων, επεξεργάζεται έγγραφα έως 2 GB χωρίς να φορτώνει ολόκληρο το αρχείο στη μνήμη, και παρέχει SLA διαθεσιμότητας 99,9 %, καθιστώντας το αξιόπιστη επιλογή για εφαρμογές επιχειρηματικού επιπέδου.

## Πώς να Ορίσετε Άδεια στο GroupDocs.Watermark για Java;

Η κλάση `License` αντιπροσωπεύει μια άδεια GroupDocs.Watermark και ενεργοποιεί τη πλήρη λειτουργικότητα του προϊόντος όταν εφαρμόζεται. Φορτώστε το αρχείο άδειας με `new License().setLicense("path/to/license.lic")` ή παρέχετε ένα `InputStream` στη μέθοδο `setLicense(InputStream)`. Αυτό το μοναδικό βήμα ενεργοποιεί ολόκληρο το API, αφαιρεί τα υδατογραφήματα αξιολόγησης και ξεκλειδώνει όλες τις premium λειτουργίες όπως η απόδοση υψηλής ανάλυσης και η επεξεργασία παρτίδων. Η μέθοδος λειτουργεί το ίδιο είτε η άδεια είναι αποθηκευμένη τοπικά, ενσωματωμένη σε πόρους ή ανακτάται από απομακρυσμένη υπηρεσία.
