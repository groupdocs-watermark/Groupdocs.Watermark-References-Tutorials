# Extraire les pièces jointes d’e‑mail avec GroupDocs.Watermark Java

Dans ce guide complet, vous découvrirez **comment extraire les pièces jointes d’e‑mail** à partir de messages au format Outlook, ajouter des filigranes et manipuler le contenu intégré à l’aide de GroupDocs.Watermark pour Java. Que vous construisiez un système sécurisé de distribution de documents ou que vous deviez automatiser des contrôles de conformité, ces tutoriels vous accompagnent pas à pas à travers des scénarios réels.

## Réponses rapides
- **Que puis‑je faire avec GroupDocs.Watermark pour Java ?** Extraire, filigraner et modifier les pièces jointes d’e‑mail et les médias intégrés.  
- **Quelle tâche principale ce guide aborde‑t‑il ?** L’extraction des pièces jointes d’e‑mail à partir de fichiers .msg, .eml et d’autres formats d’e‑mail.  
- **Ai‑je besoin d’une licence pour essayer les exemples ?** Une licence temporaire suffit pour le développement et les tests.  
- **Puis‑je traiter des fichiers PDF, Excel ou Word à l’intérieur d’un e‑mail ?** Oui – l’API gère la plupart des types de documents courants.  
- **Java 8 ou une version plus récente est‑elle requise ?** La bibliothèque prend en charge Java 8+ et fonctionne avec les builds Maven/Gradle.

## Qu’est‑ce que « extraire les pièces jointes d’e‑mail » ?
Extraire les pièces jointes d’e‑mail signifie lire programmétiquement un fichier d’e‑mail (par ex. *.msg* ou *.eml*) et récupérer chaque document joint – PDF, feuilles de calcul, images, etc. – afin de les stocker, filigraner ou analyser indépendamment du message d’origine.

## Pourquoi extraire les pièces jointes d’e‑mail avec GroupDocs.Watermark ?
- **Sécurité & image de marque** – Ajoutez des filigranes avant de transmettre ou d’archiver.  
- **Automatisation** – Traitez par lots des milliers de messages sans effort manuel.  
- **Conformité** – Assurez‑vous que les données sensibles sont marquées ou supprimées selon la politique.  
- **Flexibilité** – Fonctionne avec un large éventail de formats de pièces jointes, y compris PDF, Excel et images.

## Foire aux questions

**Q : Puis‑je extraire les pièces jointes de fichiers e‑mail chiffrés ?**  
R : Oui. Fournissez le mot de passe lors du chargement de l’e‑mail avec l’objet `EmailLoadOptions`.

**Q : La bibliothèque prend‑elle en charge le traitement en masse de milliers d’e‑mails ?**  
R : Absolument. Utilisez les API de streaming et traitez les e‑mails par lots pour limiter l’utilisation de la mémoire.

**Q : Comment ajouter un filigrane à une pièce jointe PDF extraite ?**  
R : Chargez le PDF avec `Watermarker` après l’extraction, puis appelez `addWatermark()` avec les paramètres souhaités.

**Q : Est‑il possible de supprimer des pièces jointes spécifiques tout en conservant d’autres ?**  
R : Oui. Après le chargement de l’e‑mail, parcourez `email.getAttachments()` et retirez uniquement les éléments indésirables.
