# Définir la licence GroupDocs Java – Guide de licence Watermark

Si vous intégrez **GroupDocs.Watermark** dans une application Java, la première étape consiste à **définir correctement la licence GroupDocs Java**. Sans licence valide, l'API fonctionne en mode d'essai, ce qui limite les fonctionnalités et ajoute des filigranes aux documents traités. Ce guide explique pourquoi la licence est importante, décrit les avantages d'une configuration correcte et vous dirige vers les tutoriels détaillés dont vous avez besoin pour démarrer rapidement.

## Pourquoi définir la licence GroupDocs Java est‑il important ?

Définir la licence débloque non seulement l'ensemble complet des fonctionnalités — telles que le rendu de filigranes haute résolution, le traitement par lots et la prise en charge d'un large éventail de formats de fichiers — mais garantit également le respect des conditions de licence de GroupDocs. Une licence correcte améliore les performances, supprime les filigranes d'essai et vous donne accès au support premium.
