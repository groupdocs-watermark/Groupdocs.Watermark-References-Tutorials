# GroupDocs.Watermark pour Java – Tutoriels de licence et de configuration

GroupDocs.Watermark pour Java est une bibliothèque puissante qui permet aux développeurs d’ajouter, de modifier et de gérer des filigranes sur divers formats de documents de manière programmatique. Elle prend en charge plus de 70 formats de fichiers, traite des documents jusqu’à 2 Go sans charger le fichier complet en mémoire, et offre un SLA de disponibilité de 99,9 %, ce qui en fait un choix fiable pour les applications de niveau entreprise.

## Comment définir la licence dans GroupDocs.Watermark pour Java ?

La classe `License` représente une licence GroupDocs.Watermark et active la fonctionnalité complète du produit lorsqu’elle est appliquée. Chargez votre fichier de licence avec `new License().setLicense("path/to/license.lic")` ou fournissez un `InputStream` à `setLicense(InputStream)`. Cette étape unique active l’API complète, supprime les filigranes d’évaluation et débloque toutes les fonctionnalités premium telles que le rendu haute résolution et le traitement par lots. La méthode fonctionne de la même manière que la licence soit stockée localement, intégrée aux ressources ou récupérée depuis un service distant.
