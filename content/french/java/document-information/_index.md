## Extraire la taille des pages PDF avec GroupDocs.Watermark pour Java

GroupDocs.Watermark rend simple la lecture des dimensions des pages d'un PDF sans ouvrir le fichier dans un visualiseur. En extrayant la taille des pages dès le départ, vous pouvez choisir le placement optimal du filigrane, mettre à l'échelle les graphiques de manière appropriée ou générer des rapports basés sur la taille. Les tutoriels suivants vous guident à travers chaque étape — de l'extraction basique des métadonnées à la gestion de documents complexes avec plusieurs sections.
