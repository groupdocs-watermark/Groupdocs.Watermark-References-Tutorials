# Установка лицензии GroupDocs Java – Руководство по лицензированию Watermark

Если вы интегрируете **GroupDocs.Watermark** в Java‑приложение, первым шагом является правильная **установка лицензии GroupDocs Java**. Без действующей лицензии API работает в пробном режиме, что ограничивает функциональность и добавляет водяные знаки к обрабатываемым документам. Это руководство объясняет, почему лицензирование важно, описывает преимущества правильной конфигурации и направляет вас к подробным учебникам, которые помогут быстро начать работу.

## Почему важно установить лицензию GroupDocs Java?

Установка лицензии не только открывает полный набор функций — такие как рендеринг водяных знаков высокого разрешения, пакетная обработка и поддержка широкого спектра форматов файлов — но и обеспечивает соблюдение условий лицензирования GroupDocs. Правильное лицензирование повышает производительность, удаляет пробные водяные знаки и предоставляет доступ к премиум‑поддержке.
