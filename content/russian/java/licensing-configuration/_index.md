# GroupDocs.Watermark for Java Лицензирование и Руководства по Конфигурации

## Как установить лицензию в GroupDocs.Watermark for Java?

Класс `License` представляет лицензию GroupDocs.Watermark и активирует полную функциональность продукта при применении. Загрузите файл лицензии с помощью `new License().setLicense("path/to/license.lic")` или передайте `InputStream` в `setLicense(InputStream)`. Этот единственный шаг активирует полный API, удаляет пробные водяные знаки и открывает все премиум‑функции, такие как рендеринг высокого разрешения и пакетная обработка. Метод работает одинаково, независимо от того, хранится ли лицензия локально, встроена в ресурсы или получена из удалённого сервиса.
