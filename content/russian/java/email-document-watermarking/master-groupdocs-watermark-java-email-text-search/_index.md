## Руководство по реализации

### Функция 1: Поиск текста в теле письма
Эта функция позволяет искать конкретный текст в теме и теле письма.

#### Обзор
Мы покажем, как настроить GroupDocs.Watermark для поиска по различным частям сообщения электронной почты с помощью Java‑кода.

##### Шаг 1: Определение путей к документам
Настройте пути ввода и вывода для обработки писем:
```java
String inputDocumentPath = "YOUR_DOCUMENT_DIRECTORY/message.msg";
String outputDocumentPath = "YOUR_OUTPUT_DIRECTORY/output_message.msg";
```

##### Шаг 2: Настройка параметров загрузки и Watermarker
Инициализируйте `EmailLoadOptions` и `Watermarker` для работы с вашим документом письма.
```java
EmailLoadOptions loadOptions = new EmailLoadOptions();
Watermarker watermarker = new Watermarker(inputDocumentPath, loadOptions);
```

##### Шаг 3: Создание критериев поиска
Определите критерии поиска текста. Мы будем использовать поиск без учёта регистра для ключевого слова **"test"**:
```java
import com.groupdocs.watermark.search.SearchCriteria;
import com.groupdocs.watermark.search.TextSearchCriteria;

SearchCriteria criteria = new TextSearchCriteria("test", false);
```

##### Шаг 4: Указание мест поиска
Настройте поиск, чтобы охватить как тему, так и тело писем:
```java
watermarker.getSearchableObjects().setEmailSearchableObjects(
    EmailSearchableObjects.Subject |
    EmailSearchableObjects.HtmlBody | 
    EmailSearchableObjects.PlainTextBody);
```

##### Шаг 5: Выполнение поиска и очистка водяных знаков
Выполните поиск и удалите найденные водяные знаки:
```java
import com.groupdocs.watermark.search.PossibleWatermarkCollection;

PossibleWatermarkCollection watermarks = watermarker.search(criteria);
watermarks.clear();
```

##### Шаг 6: Сохранение изменений
После обработки сохраните изменения в новый документ:
```java
watermarker.save(outputDocumentPath);
// Close the Watermarker instance to release resources
watermarker.close();
```

#### Советы по устранению неполадок
- **Распространённая проблема:** Если результаты поиска пусты, убедитесь, что текст присутствует в теле письма или в теме.  
- **Совет по производительности:** Оптимизируйте, ограничивая поиск только теми разделами, которые действительно нужны.

## Как искать несколько ключевых слов в письме
Если необходимо найти несколько терминов за один проход, создайте несколько объектов `SearchCriteria` и объедините их с помощью логических операторов **OR** или **AND**. API позволяет цепочкой соединять критерии, так что вы можете искать “invoice” **or** “receipt” без запуска отдельных циклов.

## Как удалить водяные знаки из письма
После того как вы обнаружите водяные знаки (или любой текст, соответствующий вашим критериям), метод `PossibleWatermarkCollection.clear()` удалит их из документа письма. Это именно тот шаг, который мы использовали в **Step 5** выше, и он работает для любого количества совпадений.
