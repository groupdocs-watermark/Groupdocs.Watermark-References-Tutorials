- **Primary focus:** как эффективно добавить водяной знак в книги Excel.

## Why Use GroupDocs.Watermark for Excel Watermarking?

- **Cross‑platform reliability:** Работает на Windows, Linux и macOS без дополнительных нативных зависимостей.  
- **Full‑feature support:** Обрабатывает файлы `.xls`, `.xlsx` и файлы с макросами, сохраняя формулы, диаграммы и проверки данных.  
- **Security‑first design:** Водяные знаки внедряются таким образом, что их трудно удалить без повреждения документа.
