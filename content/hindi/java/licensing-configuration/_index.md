# GroupDocs.Watermark for Java लाइसेंसिंग और कॉन्फ़िगरेशन ट्यूटोरियल्स

GroupDocs.Watermark for Java एक शक्तिशाली लाइब्रेरी है जो डेवलपर्स को विभिन्न दस्तावेज़ फ़ॉर्मेट्स पर प्रोग्रामेटिक रूप से वॉटरमार्क जोड़ने, संपादित करने और प्रबंधित करने में सक्षम बनाती है। यह 70 से अधिक फ़ाइल फ़ॉर्मेट्स का समर्थन करती है, 2 GB तक के दस्तावेज़ों को पूरी फ़ाइल को मेमोरी में लोड किए बिना प्रोसेस करती है, और 99.9 % अपटाइम SLA प्रदान करती है, जिससे यह एंटरप्राइज़‑ग्रेड एप्लिकेशनों के लिए एक विश्वसनीय विकल्प बनती है।

## GroupDocs.Watermark for Java में लाइसेंस कैसे सेट करें?

`License` क्लास GroupDocs.Watermark लाइसेंस को दर्शाता है और लागू होने पर पूर्ण उत्पाद कार्यक्षमता को सक्रिय करता है। अपने लाइसेंस फ़ाइल को `new License().setLicense("path/to/license.lic")` के साथ लोड करें या `setLicense(InputStream)` के लिए एक `InputStream` प्रदान करें। यह एकल कदम पूर्ण API को सक्रिय करता है, इवैल्यूएशन वॉटरमार्क हटाता है, और हाई‑रेज़ोल्यूशन रेंडरिंग तथा बैच प्रोसेसिंग जैसे सभी प्रीमियम फीचर्स को अनलॉक करता है। यह मेथड उसी तरह काम करता है चाहे लाइसेंस स्थानीय रूप से संग्रहीत हो, रिसोर्सेज़ में एम्बेडेड हो, या रिमोट सर्विस से प्राप्त हो।
