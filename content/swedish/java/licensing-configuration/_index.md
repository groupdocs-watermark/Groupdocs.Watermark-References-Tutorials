# GroupDocs.Watermark för Java Licensiering och Konfigurationshandledningar

GroupDocs.Watermark for Java är ett kraftfullt bibliotek som gör det möjligt för utvecklare att lägga till, redigera och hantera vattenstämplar i olika dokumentformat programatiskt. Det stödjer över 70 filformat, bearbetar dokument upp till 2 GB utan att ladda hela filen i minnet, och erbjuder en 99.9 % driftstid‑SLA, vilket gör det till ett pålitligt val för företagsapplikationer.

## Hur man anger licens i GroupDocs.Watermark för Java?

`License`-klassen representerar en GroupDocs.Watermark-licens och aktiverar hela produktens funktionalitet när den tillämpas. Ladda din licensfil med `new License().setLicense("path/to/license.lic")` eller tillhandahåll en `InputStream` till `setLicense(InputStream)`. Detta enkla steg aktiverar hela API:et, tar bort utvärderingsvattenstämplar och låser upp alla premiumfunktioner såsom högupplöst rendering och batch‑bearbetning. Metoden fungerar på samma sätt oavsett om licensen lagras lokalt, är inbäddad i resurser eller hämtas från en fjärrtjänst.
