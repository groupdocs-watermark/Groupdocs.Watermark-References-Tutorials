## Vanliga frågor

**Q: Hur lägger jag till en enkel textvattenstämpel i en PDF?**  
A: Använd klassen `WatermarkOptions` för att definiera text, typsnitt, storlek, färg och position, och applicera den sedan med metoden `Watermarker.add`.

**Q: Kan jag lägga till en bildvattenstämpel med transparens?**  
A: Ja, API‑et låter dig ladda en bild, sätta opacitetsnivån och placera den var som helst på sidan.

**Q: Är det möjligt att låsa en vattenstämpel så att användare inte kan ta bort den?**  
A: Absolut—aktivera låsningsfunktionen för vattenstämpeln när du skapar den för att göra den manipulationsresistent.

**Q: Stöder GroupDocs.Watermark lösenordsskyddade PDF‑filer?**  
A: Ja, du kan öppna en skyddad PDF genom att ange lösenordet i `LoadOptions` innan du applicerar vattenstämplar.

**Q: Vilka Java‑versioner är kompatibla med GroupDocs.Watermark?**  
A: Biblioteket stödjer Java 8 och nyare, inklusive Java 11, 17 och senare LTS‑utgåvor.
