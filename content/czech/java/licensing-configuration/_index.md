# Nastavte licenci GroupDocs Java – Průvodce licencováním Watermark

Pokud integrujete **GroupDocs.Watermark** do Java aplikace, prvním krokem je **správně nastavit licenci GroupDocs Java**. Bez platné licence API běží v režimu zkušební verze, což omezuje funkčnost a přidává vodoznaky do zpracovaných dokumentů. Tento průvodce vysvětluje, proč je licencování důležité, popisuje výhody správné konfigurace a odkazuje vás na podrobné tutoriály, které vám pomohou rychle začít.

## Proč je nastavení licence GroupDocs Java důležité?

Nastavení licence nejen odemyká kompletní sadu funkcí – například renderování vodoznaků ve vysokém rozlišení, dávkové zpracování a podporu široké škály formátů souborů – ale také zajišťuje soulad s licenčními podmínkami GroupDocs. Správné licencování zlepšuje výkon, odstraňuje zkušební vodoznaky a poskytuje vám přístup k prémiové podpoře.
