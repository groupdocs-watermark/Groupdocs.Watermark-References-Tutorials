## Najczęściej zadawane pytania

**Q: Jak dodać prosty tekstowy znak wodny do PDF?**  
A: Użyj klasy `WatermarkOptions`, aby określić tekst, czcionkę, rozmiar, kolor i pozycję, a następnie zastosuj go metodą `Watermarker.add`.

**Q: Czy mogę dodać graficzny znak wodny z przezroczystością?**  
A: Tak, API umożliwia wczytanie obrazu, ustawienie poziomu nieprzezroczystości oraz pozycjonowanie go dowolnie na stronie.

**Q: Czy istnieje możliwość zablokowania znaku wodnego, aby użytkownicy nie mogli go usunąć?**  
A: Oczywiście — włącz funkcję blokowania znaku wodnego podczas jego tworzenia, aby uczynić go odpornym na manipulacje.

**Q: Czy GroupDocs.Watermark obsługuje PDF zabezpieczone hasłem?**  
A: Tak, możesz otworzyć zabezpieczony PDF, podając hasło w `LoadOptions` przed zastosowaniem znaków wodnych.

**Q: Jakie wersje Javy są kompatybilne z GroupDocs.Watermark?**  
A: Biblioteka obsługuje Java 8 i nowsze, w tym Java 11, 17 oraz późniejsze wydania LTS.
