# Samouczki dotyczące licencjonowania i konfiguracji GroupDocs.Watermark dla Javy

GroupDocs.Watermark for Java to potężna biblioteka, która umożliwia programistom dodawanie, edytowanie i zarządzanie znakami wodnymi w różnych formatach dokumentów programowo. Obsługuje ponad 70 formatów plików, przetwarza dokumenty do 2 GB bez ładowania całego pliku do pamięci i zapewnia 99,9 % dostępności SLA, co czyni ją niezawodnym wyborem dla aplikacji klasy korporacyjnej.

## Jak ustawić licencję w GroupDocs.Watermark dla Javy?

Klasa `License` reprezentuje licencję GroupDocs.Watermark i aktywuje pełną funkcjonalność produktu po zastosowaniu. Załaduj plik licencji za pomocą `new License().setLicense("path/to/license.lic")` lub podaj `InputStream` do `setLicense(InputStream)`. Ten pojedynczy krok aktywuje pełne API, usuwa znaki wodne wersji demonstracyjnej i odblokowuje wszystkie funkcje premium, takie jak renderowanie w wysokiej rozdzielczości oraz przetwarzanie wsadowe. Metoda działa tak samo, niezależnie od tego, czy licencja jest przechowywana lokalnie, wbudowana w zasoby, czy pobierana z usługi zdalnej.
